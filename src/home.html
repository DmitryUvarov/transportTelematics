Идеально подойдет для небольших транспортных предприятий и таксопарков разных размеров.

Позволяет отображать местоположение объекта и его параметры на карте в реальном времени. Удобен для диспетчеров и логистов, когда нужно оперативно знать где и какие автомобили находятся.
Функионал, позволяющий отображать маршруты следования транспорта за прошедший период времени. Присутствует фильтрация: поездки, остановки и стоянки. Стоимость напрямую зависит от необходимого срока хранения исторических данных.
Система позволяет выборочно конфигурировать требуемый набор отчетов для каждого клиента. Основные виды отчетов: по пробегу; по поездкам; по стоянкам; по превышениям скорости; по геозонам; о неисправностях; о перепробегах; о связи с объектами; о статусах выдачи ТС. 
Модуль позволяет блокировать и разблокировать ТС, а также отображать записи о всех блокировках транспортных средств с указанием типа блокировки: вручную, по API или балансу, по выезду из назначенных администратором геозон.
Модуль позволяет создавать пользовательские аккаунты для диспетчеров, логистов или менеджеров. При подключении - основной аккаунт автоматически становится административным.
Модуль позволяет настроить уведомления, которые будут приходить в интерфейс, или Telegram-канал или по E-mail по ряду событий, например таких, как превышение скоростного режима.

Система обмена сообщениями внутри системы мониторинга. Позволяет создавать тикет для других пользователей Вашего аккаунта или технической поддержки сервиса.
Модуль назначения водителей на транспортные средства. Удобен для сервисов доставки и такси, где водители регулярно сменяют друг-друга. Интегрирован с Драйвматикой.
Модуль автоматической оплаты платных парковок. Если автомобиль находится в зоне платной парковки - система самостоятельно открывает парковочную сессию и оплачивает парковку с баланса таксометра водителя. Для Москвы и Санкт-Петербурга. Временно не работает в связи с РЭБ
Модуль позволяет отслеживать количество бортов в простое, ремонте и на линии. Имеет возможность распределения ТС по колоннам (подразделениям) для наглядного отображения эффективности работы подразделений.
Документированный API для интеграций со сторонними сервисами, такими как 1С, CRM и любыми другими внешними системами.
